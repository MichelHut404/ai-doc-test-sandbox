##### `async Task<string> GetAsync(string url)`

- **Purpose:** Performs an asynchronous HTTP GET request to the specified URL and returns the response content as a string.
- **Parameters:**
  - `url` (`string`): The URL to send the GET request to.
- **Returns:** A `Task<string>` representing the asynchronous operation, with the content of the HTTP response.
- **Remarks:** The method creates an instance of `HttpClient`, sends a GET request, ensures the request was successful, and reads the response content as a string.

```csharp
public async Task<string> GetAsync(string url)
{
    using var client = new HttpClient();
    var response = await client.GetAsync(url);
    response.EnsureSuccessStatusCode();
    return await response.Content.ReadAsStringAsync();
}
```
