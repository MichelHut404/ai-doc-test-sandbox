# IAnimal

This is an interface that determines what the class that uses the interface needs to implement.
 
## Properties

- `string Name`: The name of the animal.
- `string Species`: What kind of animal it is.
- `int Age`: The age of the animal.

## Methods

These methods need to be implemented to use the interface:

- `void MakeSound()`: Makes the animal produce its sound.
- `void Eat(string food)`: Makes the animal eat the given food.
- `string GetDescription()`: Returns a description of the animal.
