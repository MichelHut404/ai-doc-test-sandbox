# Zoo.cs
This file contains the implementation of the Zoo class which simulates a zoo environment. It allows for adding animals and displaying their details when the zoo opens.

## Zoo
Represents a zoo that can hold and manage animals.

### `public Zoo(string name)`
Constructor that initializes the zoo with a given name.
- **name** (`string`): The name of the zoo.
- **Returns**: None

### `public void AddAnimal(IAnimal animal)`
Adds an animal to the zoo and logs its addition.
- **animal** (`IAnimal`): The animal to be added to the zoo.
- **Returns**: None

### `public void OpenDeuren()`
Opens the zoo to the public, displays the animals, and their respective sounds and food preferences.
- **Returns**: None

### `public void SluitDeuren()`
Closes the zoo for the day and logs a closing message.
- **Returns**: None

### `public static void Main(string[] args)`
Entry point of the program that creates a zoo, adds animals, and manages the zoo operations.
- **args** (`string[]`): Command line arguments.
- **Returns**: None
