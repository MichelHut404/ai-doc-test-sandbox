# Zoo.cs
This file contains the implementation of a Zoo class which manages a collection of animals and provides operations to add animals and interact with them.

## Zoo
Manages a collection of animals in a zoo setting.

### `public Zoo(string name)`
Constructor that initializes the zoo with a name.
- **name** (`string`): The name of the zoo.
- **Returns**: None

### `public void AddAnimal(IAnimal animal)`
Adds a new animal to the zoo's collection and prints a message confirming its addition.
- **animal** (`IAnimal`): The animal to be added to the zoo.
- **Returns**: None

### `public void OpenDeuren()`
Simulates opening the zoo by welcoming visitors and providing details about the animals.
- **Returns**: None

### `public void SluitDeuren()`
Simulates closing the zoo for the day with a farewell message.
- **Returns**: None

### `public static void Main(string[] args)`
Entry point of the program that demonstrates the usage of the Zoo class by adding animals and opening/closing the zoo.
- **args** (`string[]`): Command line arguments.
- **Returns**: None
