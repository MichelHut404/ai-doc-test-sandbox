# Zoo.cs
This file contains the definition of the Zoo class and its related functionalities, including the management of animals within the zoo.

## Zoo
Represents a zoo that can manage a collection of animals.

### `public Zoo(string name)`
Constructor that initializes a new instance of the Zoo class with a specified name.
- **name** (`string`): The name of the zoo.
- **Returns**: None

### `public void AddAnimal(IAnimal animal)`
Adds a new animal to the zoo and announces the addition.
- **animal** (`IAnimal`): The animal to be added to the zoo.
- **Returns**: None

### `public void OpenDeuren()`
Opens the zoo for visitors and displays the animals present in the zoo, along with their sounds and food preferences.
- **Returns**: None

### `public void SluitDeuren()`
Closes the zoo for the day with a closing message.
- **Returns**: None

### `public static void Main(string[] args)`
Entry point for the application that creates a zoo and populates it with animals.
- **args** (`string[]`): Command line arguments.
- **Returns**: None
