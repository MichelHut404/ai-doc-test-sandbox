# Zoo.cs
This file contains the implementation of a Zoo class which manages a collection of animals in the zoo, allowing for their addition and interaction.

## Zoo
The Zoo class represents a collection of animals and provides methods to manage and interact with them.

### `public Zoo(string name)`
Constructor that initializes the zoo with a specified name.
- **name** (`string`): The name to assign to the zoo.
- **Returns**: N/A

### `public void AddAnimal(IAnimal animal)`
Adds an animal to the zoo and prints a confirmation message.
- **animal** (`IAnimal`): The animal to be added to the zoo.
- **Returns**: N/A

### `public void OpenDeuren()`
Opens the zoo for visitors, displaying a welcome message and describing each animal, including sounds and food preferences.
- **Returns**: N/A

### `public void SluitDeuren()`
Closes the zoo for the day and displays a closing message.
- **Returns**: N/A

### `public static void Main(string[] args)`
Entry point of the application, demonstrating the creation of a zoo and adding animals to it.
- **args** (`string[]`): Command line arguments.
- **Returns**: N/A
