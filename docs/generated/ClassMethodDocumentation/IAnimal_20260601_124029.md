# IAnimal.cs
This file contains the definition of the IAnimal interface, which establishes a contract for animal behavior in the application.

## IAnimal
Defines the properties and methods that all animal classes must implement.

### `string Name { get; }`
Gets the name of the animal.
- **Returns**: The name of the animal as a string.

### `string Species { get; }`
Gets the species of the animal.
- **Returns**: The species of the animal as a string.

### `int Age { get; }`
Gets the age of the animal in years.
- **Returns**: The age of the animal as an integer.

### `void MakeSound()`
Makes the sound associated with the animal.
- **Returns**: This method does not return any value.

### `void Eat(string food)`
Feeds the animal with the specified food.
- **food** (`string`): The food item that the animal will eat.
- **Returns**: This method does not return any value.

### `string GetDescription()`
Provides a description of the animal, including its name, species, and age.
- **Returns**: A string describing the animal's attributes in a formatted manner.
